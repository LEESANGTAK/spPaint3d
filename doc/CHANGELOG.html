Version

2022.0

Compatibility

  • Python 2.5+ (From Maya 2009 - 2020)
  • Python 3.x (From Maya 2022 onward)
  • All platforms supported (Windows, Mac, Linux)

Changelog Information

2022.0

  • Simple and limited reformatting of the code from version 2011.1. The main goal was to make a version compatible with the newest Maya release which now uses by default a python version 3 runtime.
  • Removed all legacy mel scripts.
  • Added a module file to make it easier to install. See installation information in the README file.
  • Added a drag and drop installer.
  • Remains backward compatible all the way back to whichever Maya version the 2011.x version of this script was compatible with.
  • No new feature whatsoever (yet?)
